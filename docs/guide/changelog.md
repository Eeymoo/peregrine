# 更新日志

本页面记录 Peregrine 各版本的主要变更。正式版与测试版合并按时间倒序展示。

完整正式版日志见 [GitHub Releases](https://github.com/Eeymoo/peregrine/releases)。

## v0.1.4-alpha.0 — 2026-07-11

### 变更

- **协议变更为 MIT**：从 PolyForm Noncommercial 1.0.0 改为 MIT，完全开源，允许商业使用。

### 新增

- **全屏 / 窗口覆盖模式**：全屏模式下覆盖层直接覆盖整个屏幕，无需选择目标窗口；窗口模式下仅覆盖目标窗口区域。在配置页面勾选「窗口模式」或在托盘菜单切换，两侧自动同步。
- **拖拽时实时显示设置**：在「设置」中可开启「拖拽时实时显示」，开启后窗口拖拽过程中覆盖层跟随移动；关闭时（默认）停止拖拽后约 1200ms 恢复显示，降低 CPU 占用。
- **GPU 硬件加速开关**：在「设置」中可开启 GPU 硬件加速（默认关闭），关闭时使用纯 CPU 渲染以减少 GPU 进程内存占用。
- **版本号自动化**：版本号从 git tag 动态读取，CI 打包时自动同步到各处，不再手动维护。

### 优化

- **预览比例修复**：预览区现在以真实分辨率构建视觉锚点形状再等比缩放，预览所见即覆盖层所得，不再因 canvas 尺寸导致比例失真。
- **静态视觉锚点不再持续重绘**：引入脏标记机制，静止不动的视觉锚点不再每帧重绘，显著降低覆盖层 CPU 占用。
- 限制 overlay 渲染帧率为 60 FPS，消除重复渲染导致的忙循环。
- 关闭配置/设置窗口时真正销毁 WebView2，不再隐藏到托盘占用内存。
- 启动时不预创建设置窗口，按需创建进一步降低启动内存。

### 修复

- 修复全屏模式下覆盖层位置错误：首次创建时未预定位。
- 修复打开配置页面时覆盖状态显示错误：`get_overlay_active` 直接读取原子状态。
- 修复 ESC 对话框行为：ESC 取消等同于停止覆盖并关闭对话框；保持配置窗口不会停止覆盖。
- 修复托盘「退出」失效：通过 `quitting` 标志区分主动退出与窗口关闭。

## v0.1.3 — 2026-07-11（正式版）

### 新增

- 应用国际化：支持简体中文与英文，窗口标题、托盘菜单、错误提示同步切换。
- 语言设置新增「跟随系统」选项，默认根据系统语言自动选择。
- 设置页新增「开始覆盖时自动切换到游戏」偏好：每次询问 / 是 / 否。
- 首次点击「开始覆盖」时弹出确认对话框，可选择是否记住该选择。

### 优化

- 图标清晰度大幅提升：生成脚本改用 8x 超采样抗锯齿，高 DPI 下清晰锐利。
- 发布产物改为便携 zip 压缩包，下载解压即用。

### 修复

- 修复托盘菜单语言跟随系统语言失效。
- 修复「开始覆盖后自动隐藏并切换到游戏」功能失效。
- 修复配置预览棋盘格背景错乱。
- 修复 `RandomOrb` 样式前端预览与覆盖层 RNG 不一致。

## v0.1.0 — 2026-07-11（正式版）

### 新增

- Windows 透明置顶穿透覆盖层。
- 目标窗口跟随。
- 12 种视觉锚点样式（边缘矩形、十字准星、大型十字准星、四/六/八角标记、中心圆环、自定义边缘标记、随机边缘标记、边框、边缘箭头等）。
- 自定义 PNG 贴图。
- 配置热重载。
- 系统托盘集成。
