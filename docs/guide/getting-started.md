# 快速开始

## 下载安装

访问 [GitHub Releases](https://github.com/eeymoo/peregrine/releases) 页面，下载对应平台的可执行文件。

> 当前主要支持 Windows（x86 / x86_64 / ARM64）。

## 启动程序

下载后双击 `peregrine.exe` 运行。程序启动后默认进入**设置模式**，按 **Tab** 可在设置模式与覆盖层模式之间切换。

## 常用快捷键

| 按键 | 作用 |
|------|------|
| `Tab` | 切换 设置 / 覆盖层 模式 |
| `Esc` | 设置模式下返回覆盖层；覆盖层模式下退出程序 |

## 第一步：选择准心样式

1. 在设置面板中选择当前激活的 **Profile**。
2. 在 **Crosshair** 部分选择你喜欢的样式，例如 `Cross`、`Ring`、`ToiletPaper`。
3. 调整尺寸、透明度、颜色等参数。
4. 按 `Tab` 切换到覆盖层模式，即可在游戏中看到效果。

## 配置文件

配置文件位于：

- Windows：`%APPDATA%\Peregrine\config.json`

你可以直接编辑该文件，保存后程序会自动热重载。
