# 更新日志

本页面记录 Peregrine 各版本的主要变更。正式版与测试版合并按时间倒序展示。

完整正式版日志见 [GitHub Releases](https://github.com/Eeymoo/peregrine/releases)。

---

## v0.1.4 — 2026-07-11（正式版）

协议变更为 MIT 完全开源；新增全屏/窗口覆盖模式、GPU 加速开关、屏幕缩放自适应；大幅优化内存占用与 CPU 消耗。

### 变更

- **协议变更为 MIT**：从 PolyForm Noncommercial 1.0.0 改为 MIT，完全开源，允许商业使用。

### 新增

- **全屏 / 窗口覆盖模式**：全屏模式（默认）直接覆盖整个屏幕，无需选择目标窗口；窗口模式仅覆盖目标窗口区域。配置页面勾选「窗口模式」或托盘菜单切换，两侧自动同步。
- **拖拽时实时显示设置**：在「设置」中开启后窗口拖拽时覆盖层实时跟随；关闭时（默认）停止拖拽约 1200ms 后恢复显示，降低 CPU 占用。
- **GPU 硬件加速开关**：在「设置」中可开启 GPU 硬件加速（默认关闭），切换后弹出重启确认对话框。
- **版本号自动化**：版本号从 git tag 动态读取，CI 打包时自动同步。

### 优化

- **预览比例修复**：预览区以真实分辨率构建视觉锚点形状再等比缩放，所见即所得。
- **静态视觉锚点不再持续重绘**：引入脏标记机制，显著降低覆盖层 CPU 占用。
- **屏幕缩放自适应**：全屏覆盖层跟随分辨率/DPI 变化即时更新。
- **预览实时刷新**：加入 ResizeObserver，窗口拖拽/缩放时预览立即重绘。
- **配置保存防抖**：连续操作只在停止后 300ms 写入一次。
- 关闭窗口时真正销毁 WebView2，不再隐藏占内存。
- 发布产物 zip 内含 `peregrine-v*.exe`、`README.md`、`LICENSE`。

### 修复

- 修复全屏模式下覆盖层位置错误。
- 修复打开配置页面时覆盖状态显示错误。
- 修复 ESC 对话框行为。
- 修复 WebView2 进程关闭后未释放内存。
- 修复托盘「退出」失效。
- 修复文档部署 CI 失败。

## v0.1.3 — 2026-07-11（正式版）

### 新增

- 应用国际化：支持简体中文与英文，窗口标题、托盘菜单、错误提示同步切换。
- 语言设置新增「跟随系统」选项，默认根据系统语言自动选择。
- 设置页新增「开始覆盖时自动切换到游戏」偏好：每次询问 / 是 / 否。
- 首次点击「开始覆盖」时弹出确认对话框，可选择是否记住该选择。

### 优化

- 图标清晰度大幅提升：生成脚本改用 8x 超采样抗锯齿，高 DPI 下清晰锐利。
- 发布产物改为便携 zip 压缩包，下载解压即用。

### 修复

- 修复托盘菜单语言跟随系统语言失效。
- 修复「开始覆盖后自动隐藏并切换到游戏」功能失效。
- 修复配置预览棋盘格背景错乱。
- 修复 `RandomOrb` 样式前端预览与覆盖层 RNG 不一致。

## v0.1.0 — 2026-07-11（正式版）

### 新增

- Windows 透明置顶穿透覆盖层。
- 目标窗口跟随。
- 12 种视觉锚点样式（边缘矩形、十字准星、大型十字准星、四/六/八角标记、中心圆环、自定义边缘标记、随机边缘标记、边框、边缘箭头等）。
- 自定义 PNG 贴图。
- 配置热重载。
- 系统托盘集成。
