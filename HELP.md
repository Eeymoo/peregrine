# Peregrine 使用帮助

本文档面向 **使用者**,介绍 Peregrine 是什么、怎么用,以及每种辅助贴图的作用和调节方法。开发相关信息请看 [README](README.md)。

---

## 1. Peregrine 是什么

Peregrine 是一个在屏幕上绘制**半透明视觉锚点**的小工具。它的主要用途是**缓解 3D 眩晕**。

很多人在玩第一人称 / 第三人称 3D 游戏时会头晕、恶心,原因之一是画面剧烈晃动时,眼睛在屏幕上找不到一个**固定不动的参照物**。Peregrine 在屏幕的中心或边缘画出始终不动的锚点(准心、定位球、边框等),给眼睛一个稳定的落点,从而减轻眩晕。

> 你可以把它理解成"电子版贴在屏幕上的卫生纸"——很多人晕 3D 时会真的往屏幕中间贴一小块纸做参照,这也是默认样式叫"卫生纸"的原因。

---

## 2. 快速上手

1. 启动程序后,会先进入 **设置界面**,右侧是设置面板,左侧是实时预览。
2. 在"类型"下拉里选一个你喜欢的样式,拖动下面的滑块调整大小、颜色、透明度,左侧会**即时**显示效果。
3. 调好后按 **`Tab`** 键切换到**覆盖层模式**,锚点就会显示出来。
4. 想再改就按 `Tab` 切回设置界面。

### 快捷键

| 按键 | 作用 |
| --- | --- |
| `Tab` | 在「覆盖层」和「设置」之间来回切换 |
| `Esc` | 在设置界面 → 切回覆盖层;在覆盖层 → 退出程序 |

你的所有设置都会**自动保存**,下次打开还是这套配置,不用手动保存。

---

## 3. 通用设置

无论选哪种样式,这几项都能调:

- **透明度**:0 = 完全看不见,1 = 完全不透明。防眩晕通常调到**偏低**(比如 0.3~0.6),既能看见锚点又不太挡视线。
- **颜色**:点色块可自由选颜色。建议选一个和游戏画面**对比明显**的颜色(亮绿、洋红等),更容易被眼睛"抓住"。

---

## 4. 各种样式详解

在设置面板"类型"下拉里选择。每种样式的可调项不同。

### 卫生纸(默认)
屏幕中心一块半透明矩形,模拟贴了一张卫生纸。最简单直接的中心参照物。
- **宽度 / 高度**:矩形大小(10~400 / 10~300 像素)。
- **圆角**:矩形四角的圆润程度。
- **贴边 / 边距**:可以让矩形贴到屏幕某一边,而不是正中心。

### 准星
屏幕正中心一个十字线,占用面积小。
- **臂长**:十字每条线的长度。
- **线宽**:线的粗细。
- **中心间隙**:十字正中间留空的大小(留空后中心是"空心十字")。

### 大准星
从屏幕边缘一直延伸到中心的一条横线和一条竖线,把整个屏幕分成四块,参照感更强。
- **线宽**:线的粗细。

### 定位球 4 / 6 / 8
在屏幕四周放置若干圆点作为锚点,不挡中心视野,适合需要看清画面中央的场景。
- **定位球 4**:四个角各一个。
- **定位球 6**:四角 + 上下中间。
- **定位球 8**:四角 + 上下中间 + 左右中间。
- 可调**距边缘距离**、**半径**(设为 0 时按线宽自动计算)、**线宽**。

### 中心环
以屏幕中心为圆心的一个圆环,提供环形锚点。
- **半径占屏高比例**:环的大小(相对屏幕高度)。
- **线宽** 和**线型**:实线 / 虚线 / 双环。

### 自定义定位球
自己决定在上、下(左、右为预留)边缘各放几个圆点。
- 勾选启用**上 / 下 / 左 / 右**边。
- 分别设置每条边的**点数量**(1~10)、**半径**、**距边缘距离**。

### 随机球
启动时在屏幕边缘区域**随机**生成一批圆点,位置和大小每次可能不同,适合喜欢不规则参照的人。
- **每边数量、距边缘距离、位置扰动、最小/最大半径**都可调。
- 工作模式(需在配置文件中调整):
  - `lock_on_start`:第一次随机后**固定下来**,以后每次打开位置都一样。
  - `reshuffle`:**每次启动**都重新随机。

### 边框
沿屏幕四边画一圈线,像电影的"安全框",给出画面边界参照。
- **矩形条高度、距边缘距离**可调。
- **样式**:完整四边,或四边中间留缺口。
- 勾选**四边中间留 20% 缺口**:避免遮挡小地图、血条等 HUD 元素。

## 5. 多套配置(Profile)

Peregrine 支持多个 **Profile**(配置档),可以为不同游戏或场景各存一套设置,切换当前激活的 Profile 即可套用对应样式。目前 Profile 的新增/切换主要通过[配置文件](#7-配置文件)完成。

---

## 6. 防眩晕使用建议

- **先从"卫生纸"或"准星"入手**,它们最直观。
- **透明度别太高**,锚点太实反而更累;能隐约看见就够了。
- **颜色要跳**,和游戏主色调拉开对比。
- 如果晕得厉害,试试**大准星**或**边框**这种"占满屏幕"的样式,参照信息更多。
- 玩久了可以适当**缩小 / 调淡**锚点,让眼睛逐步适应。

---

## 7. 配置文件

所有设置以 JSON 保存在系统标准目录:

| 系统 | 路径 |
| --- | --- |
| macOS | `~/Library/Application Support/Peregrine/config.json` |
| Windows | `%APPDATA%/Peregrine/config.json` |
| Linux | `~/.config/Peregrine/config.json` |

- 首次运行会**自动生成**默认配置,一般不需要手动改。
- 你也可以直接用文本编辑器修改这个文件,**保存后程序会自动重新加载**(热重载),无需重启。
- 某些高级项(如随机球工作模式、Profile 管理)目前只能在配置文件里改。
- 万一配置改坏了,删掉该文件再启动程序,会重新生成一份默认配置。

## 8. 常见问题

**Q:切到覆盖层后,锚点后面是白色/不透明的,没法看游戏?**
A:当前版本的覆盖层还是一个**普通窗口**,尚未实现真正的透明、置顶和鼠标穿透。这是已知的开发中限制,后续版本会做真正的穿透式覆盖层。

**Q:"选择窗口"按钮点了没反应?**
A:"跟随目标窗口"目前是**占位功能**,还没接入系统 API,点击不会有实际效果。

**Q:设置里的"触发规则 / 进程触发"是干嘛的?**
A:计划用于"检测到某个游戏进程运行时才自动显示锚点",目前同样是**占位**,尚未生效。

**Q:改了窗口大小,锚点位置不对?**
A:当前版本不会在窗口尺寸变化后重新排布,建议保持默认窗口大小使用。

**Q:我的设置丢了/想恢复默认?**
A:关掉程序,删除上面路径里的 `config.json`,重新启动即可恢复默认配置。

---

> 说明:Peregrine 目前处于**骨架阶段**,界面与配置已可正常使用,但覆盖层的透明穿透、窗口跟随、进程触发等特性仍在开发中。遇到上述"占位"功能不生效属于预期情况。
