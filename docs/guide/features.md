# 功能特性

## 覆盖层模式

在覆盖层模式下，Peregrine 会在屏幕最上层绘制辅助贴图，窗口特性包括：

- **透明**：背景完全透明，只显示视觉锚点或贴图。
- **置顶**：始终保持在其他窗口之上。
- **鼠标穿透**：不拦截鼠标点击和键盘输入，游戏操作不受影响。
- **窗口跟随**：可选跟随指定游戏窗口移动。

## 准心样式

`CrosshairStyle` 当前支持以下样式：

| 样式 | 说明 |
|------|------|
| `EdgeRect` | 边缘矩形，经典视觉锚点 |
| `Cross` | 十字准星 |
| `LargeCross` | 大型十字准星 |
| `CornerDots4` | 四角边缘标记（4 个） |
| `CornerDots6` | 六角边缘标记（6 个） |
| `CornerDots8` | 八角边缘标记（8 个） |
| `Ring` | 中心圆环 |
| `CustomOrb` | 自定义边缘标记 |
| `RandomOrb` | 随机边缘标记 |
| `BorderFrame` | 边框 |
| `CustomImage` | 自定义图片 |
| `EdgeArrows` | 边缘箭头 |

## 自定义 PNG 贴图

你可以加载任意 PNG 图片作为辅助贴图，程序会将其解码并绘制在屏幕中心。

## 设置与预览

设置面板使用 Tauri + React 构建，提供：

- 实时参数调节
- 即时预览
- 自动保存与配置校验

## 进程触发（占位）

`TriggerRule`（按进程自动启用）已在配置模型中定义，但尚未接入平台 API，后续版本实现。
