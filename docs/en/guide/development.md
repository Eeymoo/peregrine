# Development Build

## Environment Requirements

- Rust 1.85 or later (edition 2024)
- Windows SDK (for Win32 API and the `windows` crate)
- Cargo

## Clone the Repository

```bash
git clone https://github.com/eeymoo/peregrine.git
cd peregrine
```

## Build

```bash
# Debug build
cargo build

# Release build (small size, high performance)
cargo build --release

# Run the GUI main program
cargo run -p peregrine
```

## Test

```bash
# Run all tests
cargo test

# Run only the config crate tests
cargo test -p peregrine_config
```

## Code Check

```bash
cargo fmt
cargo clippy
```

## Release Artifacts

The executable generated by `cargo build --release` is located at:

```
target/release/peregrine.exe
```

The release build profile is optimized for size and performance:

- `opt-level = "z"`
- `lto = true`
- `codegen-units = 1`
- `strip = true`
- `panic = "abort"`
